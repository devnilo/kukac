    Abaixo temos as informações necessárias para utilização dos serviços Kukac

    NÚMEROS PALÍNDROMOS

    Ao acessar nossa página de números palíndromos, você precisará informar um número inicial e um número limite, para nossa API consiga retornar os números palíndromos presentes dentro do intervalo informado.

    CALCULE SEU TROCO

    Ao acessar nossa página calcule o troco, você precisará informar o valor da compra e o valor pago em dinheiro, para nossa API retornar o valor do troco e a quantidade do mesmo em notas de 1, 10 e 100.

    CADASTRE SEU CARRO

    Ao acessar nossa página cadastre seu veículo, você precisará preencher um pequeno formulário para cadastrar o seu veículo. Nessa mesma página possuí a função de visualizar os veículos registrados no sistema.

    CADASTRE SUA MOTO

    Ao acessar nossa página cadastre sua moto, você precisará preencher um pequeno formulário para cadastrar a sua moto. Nessa mesma página possuí a função de visualizar as motos registradas no sistema.

    CONSULTE SEU CEP

    Ao acessar nossa página consulte seu CEP, você precisará informar um número de CEP real, para que nossa API consiga retornar informações como seu estado, cidade, bairro e endereço.
